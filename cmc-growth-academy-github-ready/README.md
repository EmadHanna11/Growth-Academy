## Publish on GitHub Pages

1. Create a new public GitHub repository, for example: `growth-academy`.
2. Upload all files and folders from this package to the repository root.
3. Go to Settings → Pages.
4. Under Build and deployment, choose Deploy from a branch.
5. Select branch `main` and folder `/root`, then Save.
6. Wait a few minutes and open the generated GitHub Pages link.

Important: `index.html` must stay in the repository root, not inside another folder.
